# 今天给大家推荐一款git可视化工具(GitKraken)
## 这个工具让我说怎么样？  我只想说 完美 这个工具第一是漂亮的ui界面(我不是舔狗 这个真的是很漂亮) 给大家看看
![GitKraken的操作界面](/img/git1.png)
## 第二就是超级简单的提交代码到git的操作 不需要记那些 git add . , git status , git commit -m “” , git push 等等指令 只需点一下就行 而且你不需要下载git！！！ 因为这个软件自带git是不是很强 
### 接下来我就给大家介绍他从克隆项目 然后到拉取新代码--commit--提交代码 的简单使用吧
### 1.我们需要先把项目从git上克隆下来
![1.获取github上的克隆地址](/img/git2.png)
### 2.新建一个文件夹 右键(如果你安装了GitKraken就会有这个选项)
![2.鼠标右键点击GitKraken](/img/git3.png)
### 3.点一下进去后左下角会出来这个 大家不用管
![3.进入GitKraken](/img/git4.png)
### 4.进去后按此步骤点克隆项目
![4.准备克隆](/img/git5.png)
![5.准备克隆](/img/git6.png)
### 5.然后按照这个要求把需要的东西填进去然后静待佳音
![6.开始克隆](/img/git7.png)
![6.等待克隆完成](/img/git8.png)
### 6.然后点open now就进入主页面了
![6.进入主页面](/img/git9.png)
### 7.然后开始提交自己改的东西
![7.先pull一下拉去新的代码(每次提交件之前都要pull一下 这个很重要！！！！！ 不然会冲突)](/img/git10.png)
### 8.然后点中间第一个你右面会出来要提交的内容
![9.你要提交的内容都在这](/img/git11.png)
### 9.把你要提交的内容全部放到暂存区
![10.准备放到暂存区](/img/git12.png)
![10.放到暂存区](/img/git13.png)
### 10.提交后你会发现中间的那块多了你提交的内容及备注
![10.准备提交](/img/git14.png)
### 11.最后点push 提交完成
![11.提交完成](/img/git15.png)
